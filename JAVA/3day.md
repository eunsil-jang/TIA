# 2-5 상수와 리터럴
## 변수,상수,리터럴
<br>

* 변수 (variable) 하나의 값을 저장하기 위한 공간 - 맨 처음 저장한 값을 여러번 변경할 수 있음.
* 상수 (constant) 한번만 값을 저장 가능한 변수 - 처음 저장한 값을 변경할 수 없음.
* 리터럴 (literal) 그 자체로 값을 의미하는 것
